# Indian Railways - Passenger Awareness

A modern, responsive web application designed to educate passengers about Indian Railways rules, services, penalties, and emergency procedures.

## Overview

This single-page application provides comprehensive information about Indian Railways operations, passenger guidelines, and safety measures with an engaging, animated user interface.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations** - GSAP-powered scroll animations and transitions
- **Interactive Navigation** - Fixed navbar with hamburger menu for mobile
- **Comprehensive Information** - Covers booking, rules, services, penalties, and emergencies
- **Modern UI** - Gradient backgrounds, card layouts, and hover effects

## Getting Started

1. Clone or download the project
2. Open `index.html` in a web browser
3. No build process or dependencies required

## Sections

- **Home** - Hero section with animated welcome message
- **How It Works** - Overview of Indian Railways operations
- **Rules & Regulations** - Passenger guidelines and requirements
- **Services** - Available amenities and facilities
- **Penalties** - Fines for rule violations
- **Emergency** - Safety procedures and helpline numbers

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
